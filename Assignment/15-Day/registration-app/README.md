# Registration Portal

A React-based registration form application built with React Router and Bootstrap.

## Features
- Registration form with validation (Name, Email, Phone, DOB, Gender, Address)
- Dashboard page to display registered user details
- Welcome message with username in the navbar
- Logout functionality

## How to Run

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
